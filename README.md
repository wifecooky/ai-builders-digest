# AI Builders Digest

Daily digest site tracking 25+ AI builders across X/Twitter, podcasts, and company blogs.

Generated by [digest-factory](../digest-factory). Powered by SvelteKit + Tailwind CSS.

Live: https://builders.thewang.net

## Quick Start

```bash
# Install dependencies
cd frontend && npm install

# Start dev server (http://localhost:5173)
npm run dev
```

## Generate Content

```bash
# Full pipeline: collect → remix → translate
npm run generate-daily

# Individual steps
npm run collect     # Fetch RSS feeds
npm run remix       # AI-powered summarization
npm run translate   # Translate to en/zh/ja
```

## Project Structure

```
├── site.yaml          # Site config (theme, labels, sources)
├── frontend/          # SvelteKit frontend
│   └── src/routes/    # Pages and RSS/sitemap
├── scripts/           # Data pipeline
│   ├── collect-feeds.js
│   ├── remix-ai.js
│   └── translate-ai.js
├── content/           # Generated content (en/zh/ja)
└── config/            # Editorial prompts
```
